# Οδηγίες Εγκατάστασης (Για Συμφοιτητές)

Αυτός ο οδηγός θα σε βοηθήσει να τρέξεις τον **Βοηθό Σπουδών ΕΑΠ** στον υπολογιστή σου εύκολα, χρησιμοποιώντας **Docker Compose** για τοπική ανάπτυξη.

## Προαπαιτούμενα
1.  Πρέπει να έχεις εγκατεστημένο το **Docker Desktop**.
    - [Κατέβασμα για Windows/Mac/Linux](https://www.docker.com/products/docker-desktop/)

## 🚀 Πώς να το τρέξεις (3 Βήματα με Docker Compose)

Βεβαιώσου ότι βρίσκεσαι στον φάκελο όπου υπάρχει το αρχείο `docker-compose.yml` (και το `Dockerfile`).

1.  Αποσυμπίεσε τον φάκελο με τα αρχεία του project.
2.  Άνοιξε ένα τερματικό (Terminal, PowerShell ή CMD) μέσα σε αυτόν τον φάκελο.
3.  Τρέξε την εξής εντολή για να δημιουργήσεις και να ξεκινήσεις την εφαρμογή:

    ```bash
    docker-compose up -d --build
    ```

    *Σημείωση:* Το `--build` είναι απαραίτητο μόνο την πρώτη φορά που τρέχεις την εφαρμογή ή όταν έχεις κάνει αλλαγές στον κώδικα/Dockerfile.

## 🌐 Πώς να το χρησιμοποιήσεις

1.  Άνοιξε τον browser σου (Chrome, Firefox, Edge, κλπ).
2.  Πήγαινε στη διεύθυνση: **[http://localhost:8080](http://localhost:8080)**
3.  Η εφαρμογή είναι έτοιμη για χρήση!

## ⚠️ Σημείωση για τα PDF (Πολύ Σημαντικό)
Η εφαρμογή **δεν** περιέχει τα PDF των μαθημάτων (για λόγους πνευματικών δικαιωμάτων). Όταν η εφαρμογή σου δώσει την προτεινόμενη εντολή (Prompt), θυμήσου να **επισυνάψεις (Drag & Drop)** το PDF του μαθήματος που διαβάζεις από τον δικό σου υπολογιστή στο AI Chatbot (π.χ. ChatGPT, Google Gemini, Claude) που χρησιμοποιείς.

## 🛑 Τερματισμός της Εφαρμογής

Για να σταματήσεις την εφαρμογή (και να αφαιρέσεις τα containers και τα δίκτυα που δημιουργήθηκαν), τρέξε:

```bash
docker-compose down
```
