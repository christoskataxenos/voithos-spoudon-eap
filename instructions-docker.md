# Οδηγίες Εγκατάστασης (Για Συμφοιτητές)

Αυτός ο οδηγός θα σε βοηθήσει να τρέξεις τον **Βοηθό Σπουδών ΕΑΠ** στον υπολογιστή σου εύκολα, χρησιμοποιώντας Docker.

## Προαπαιτούμενα
1.  Πρέπει να έχεις εγκατεστημένο το **Docker Desktop**.
    - [Κατέβασμα για Windows/Mac/Linux](https://www.docker.com/products/docker-desktop/)

## Πώς να το τρέξεις (3 Βήματα)

1.  Αποσυμπίεσε τον φάκελο με τα αρχεία.
2.  Άνοιξε ένα τερματικό (Terminal ή CMD) μέσα στον φάκελο.
3.  Τρέξε την εξής εντολή:

    ```bash
    docker-compose up -d --build
    ```

    *(Το `--build` χρειάζεται μόνο την πρώτη φορά)*

## Πώς να το χρησιμοποιήσεις

1.  Άνοιξε τον browser σου (Chrome, Firefox, κλπ).
2.  Πήγαινε στη διεύθυνση: **[http://localhost:8080](http://localhost:8080)**
3.  Η εφαρμογή είναι έτοιμη!

## Σημείωση για τα PDF
Η εφαρμογή **δεν** περιέχει τα PDF των μαθημάτων (για λόγους πνευματικών δικαιωμάτων). Όταν η εφαρμογή σου δώσει την εντολή (Prompt), θυμήσου να **επισυνάψεις** το PDF του μαθήματος που διαβάζεις από τον δικό σου υπολογιστή στο Google AI Studio.

## Τερματισμός

Για να σταματήσεις την εφαρμογή, τρέξε:

```bash
docker-compose down
```
