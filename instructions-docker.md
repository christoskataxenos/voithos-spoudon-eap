# 🐳 Οδηγός Εγκατάστασης με Docker
### (The Modern Way)

Αυτός ο οδηγός θα σε βοηθήσει να τρέξεις τον **Βοηθό Σπουδών ΕΑΠ** στον υπολογιστή σου χωρίς να χρειάζεται να εγκαταστήσεις Node.js, Python ή να κάνεις τάματα στον Άγιο Παΐσιο.

---

## ✅ Προαπαιτούμενα

Το μόνο που χρειάζεσαι είναι το **Docker Desktop**.
Αν δεν το έχεις, κατέβασέ το και κάνε εγκατάσταση (Next -> Next -> Finish).

📥 **[Download Docker Desktop](https://www.docker.com/products/docker-desktop)**

🔗 **[View on Docker Hub](https://hub.docker.com/r/christosk89/voithos-spoudon-eap)**

> 💡 **Tip:** Αφού το εγκαταστήσεις, βεβαιώσου ότι το **τρέχεις** (να φαίνεται το εικονίδιο του φαλαινόψαρου κάτω δεξιά ή πάνω στη μπάρα).

---

## 🛠️ Πώς να το τρέξεις (Step-by-Step)

### Βήμα 1: Προετοιμασία
1.  Αποσυμπίεσε (Unzip) τον φάκελο με τα αρχεία που κατέβασες.
2.  Μπες μέσα στον φάκελο.

### Βήμα 2: Άνοιγμα Τερματικού
Πρέπει να ανοίξεις εντολές **μέσα** στον φάκελο του project.

* 🪟 **Windows:** Μπες στον φάκελο, γράψε `cmd` ή `powershell` στη μπάρα διεύθυνσης (εκεί που λέει το path) και πάτα Enter.
* 🍎 **Mac / 🐧 Linux:** Άνοιξε Terminal και πήγαινε στον φάκελο (`cd path/to/folder`).

### Βήμα 3: Εκκίνηση (Launch)
Αντέγραψε και επικόλλησε την παρακάτω εντολή (η μοντέρνα έκδοση):

```bash
docker compose up -d --build
```

*   `up`: Ξύπνα τα containers.
*   `-d`: Detached mode (τρέξε στο παρασκήνιο και μην μου πρήζεις το τερματικό).
*   `--build`: Χτίσε την εικόνα ξανά αν έχουμε αλλάξει κώδικα.

*(Αν σου βγάλει error ότι δεν υπάρχει το `docker compose`, δοκίμασε την παλιά εντολή `docker-compose up...` αλλά μάλλον πρέπει να κάνεις update το Docker σου).*

## 🌐 Πώς να το χρησιμοποιήσεις
Μόλις τελειώσει η διαδικασία, άνοιξε τον Browser σου και πήγαινε:

👉 **[http://localhost:8080](http://localhost:8080)**

## ⚠️ ΣΗΜΑΝΤΙΚΟ (Διάβασέ το!)
Η εφαρμογή **ΔΕΝ** περιέχει τα PDF/Books του ΕΑΠ. Θα σε βοηθήσει να φτιάξεις το Prompt. Για να δουλέψει το AI, πρέπει:

1.  Να πάρεις το Prompt από την εφαρμογή.
2.  Να το βάλεις στο ChatGPT/Gemini.
3.  Να ανεβάσεις **ΕΣΥ** το αρχείο (PDF/PPT) της ύλης εκείνη τη στιγμή.

## 🛑 Πώς το σταματάω;
Για να μην τρέχει τσάμπα στο background και σου τρώει μνήμη, όταν τελειώσεις το διάβασμα τρέξε στο τερματικό:

```bash
docker compose down
```
