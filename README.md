> Ένας έξυπνος οδηγός prompts για τους ταλαιπωρημένους φοιτητές της Πληροφορικής στο Ελληνικό Ανοικτό Πανεπιστήμιο (ΕΑΠ).

Αυτό το εργαλείο είναι εδώ για να σε βοηθήσει να δημιουργήσεις **τέλειες εντολές (prompts)** για το ChatGPT / Google Gemini / Claude, μετατρέποντας το AI στον προσωπικό σου καθηγητή (που δεν βαριέται να απαντήσει).

---

## ✨ Χαρακτηριστικά

*   **📚 19 Ενότητες:** Πλήρης υποστήριξη για τα βασικά μαθήματα Πληροφορικής (Από ΠΛΗ10 μέχρι ΠΛΗΨΙ).
*   **🧠 Εξειδικευμένο Context:** Οι εντολές δεν ζητούν απλά "λύσε μου την άσκηση", αλλά εστιάζουν στην **κατανόηση** και την εξήγηση (για να μην κοπείς στην εξέταση).
*   **🇬🇷 Πλήρως Ελληνικά:** Διεπαφή και παραγόμενα prompts στη φυσική μας γλώσσα.
*   **🔒 Local & Private:** Τρέχει τοπικά στον υπολογιστή σου. Κανένα δεδομένο δεν φεύγει από το PC σου.
*   **📂 PDF Reminder:** Σου υπενθυμίζει διακριτικά ότι χωρίς το PDF της ύλης, το AI δεν ξέρει τι του γίνεται.

## 🚀 Πώς να το τρέξεις

Έχεις δύο επιλογές, ανάλογα με το πόσο hacker νιώθεις σήμερα.

### 🐳 Επιλογή 1: Με Docker (Προτείνεται)

Εάν δεν έχεις ανεβάσει το image στο Docker Hub (ή προτιμάς να το χτίσεις τοπικά):

```bash
docker build -t voithos-spoudon-eap .
docker run -p 8080:80 voithos-spoudon-eap
```

Εναλλακτικά, μπορείς να κατεβάσεις και να τρέξεις το έτοιμο image απευθείας από το Docker Hub:

```bash
docker run -p 8080:80 christosk89/voithos-spoudon-eap
```
(Δείτε το στο [Docker Hub](https://hub.docker.com/r/christosk89/voithos-spoudon-eap))

Μετά άνοιξε τον browser σου στο: [http://localhost:8080](http://localhost:8080)

### 💻 Επιλογή 2: Τοπικά (Για απλούς θνητούς)

Δεν θες να μπλέκεις με containers; Κανένα πρόβλημα.

1.  Κατέβασε αυτό το repository (Code -> Download ZIP).
2.  Αποσυμπίεσε τον φάκελο.
3.  Άνοιξε το αρχείο `index.html` με διπλό κλικ (θα ανοίξει στον Chrome/Firefox/Edge).

## 📚 Πώς λειτουργεί (Tutorial για αρχάριους)

*   **Επιλογή:** Διαλέγεις την Ενότητα που σε βασανίζει (π.χ. ΠΛΗ24).
*   **Στόχος:** Επιλέγεις τι θες να κάνεις (Εργασία; Διάβασμα; Απλή Εξήγηση;).
*   **Magic:** Το εργαλείο σου φτιάχνει μια "μαγική εντολή" (Prompt).
*   **Copy:** Πατάς το κουμπί Αντιγραφή.
*   **Paste & Upload:** Πας στο ChatGPT/Gemini, κάνεις Επικόλληση και σέρνεις μέσα (Drag & Drop) το PDF του μαθήματος.
