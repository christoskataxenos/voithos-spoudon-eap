# 🎓 Βοηθός Σπουδών ΕΑΠ (Voithos-Spoudon-EAP)

Ένας έξυπνος οδηγός prompts για φοιτητές της Πληροφορικής στο **Ελληνικό Ανοικτό Πανεπιστήμιο (ΕΑΠ)**.

Αυτό το εργαλείο σε βοηθάει να δημιουργήσεις **τέλειες εντολές (prompts)** για το ChatGPT / Google Gemini, ώστε να λειτουργούν ως ο προσωπικός σου καθηγητής.

## ✨ Χαρακτηριστικά

*   **19 Ενότητες**: Υποστήριξη για όλα τα μαθήματα Πληροφορικής (ΠΛΗ10 - ΠΛΗΨΙ).
*   **Εξειδικευμένο Context**: Οι εντολές είναι ρυθμισμένες να ζητούν **κατανόηση** και όχι έτοιμες λύσεις.
*   **Πλήρως Ελληνικά**: Διεπαφή και παραγόμενες εντολές στα Ελληνικά.
*   **Local & Private**: Τρέχει τοπικά στον υπολογιστή σου.
*   **Υποστήριξη PDF**: Σου θυμίζει ποιο PDF να ανεβάσεις για κάθε μάθημα.

## 🚀 Πώς να το τρέξεις

### Επιλογή 1: Με Docker (Προτείνεται)

Δεν χρειάζεται να κατεβάσεις κώδικα. Απλά τρέξε στο τερματικό σου:

```bash
docker run -p 8080:80 YOUR_USERNAME/voithos-spoudon-eap
```

Μετά άνοιξε το browser στο: `http://localhost:8080`

### Επιλογή 2: Τοπικά (Χωρίς Docker)

1.  Κατέβασε αυτό το repository (Code -> Download ZIP).
2.  Αποσυμπίεσε τον φάκελο.
3.  Άνοιξε το αρχείο `index.html` με τον Chrome ή Firefox.

## 📚 Πώς λειτουργεί

1.  Επιλέγεις την **Ενότητα** (π.χ. ΠΛΗ24).
2.  Επιλέγεις τι θες να κάνεις (Εργασία, Διάβασμα, Εξήγηση).
3.  Το εργαλείο σου φτιάχνει μια **μαγική εντολή**.
4.  Την κάνεις **Αντιγραφή**.
5.  Πας στο ChatGPT/Gemini, κάνεις **Επικόλληση** και σέρνεις μέσα το **PDF** του μαθήματος.

## ⚠️ Προσοχή στα PDF

Αυτό το repository **ΔΕΝ** περιέχει τα βιβλία ή τα PDF του ΕΑΠ για λόγους πνευματικών δικαιωμάτων. Πρέπει να χρησιμοποιείς τα δικά σου αρχεία που έχεις κατεβάσει από το learn.eap.gr.
